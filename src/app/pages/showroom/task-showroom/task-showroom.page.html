Aufgabentypen Navigationsaufgaben

Navigationsaufgabe verlangen das Navigieren zu einem neuen Ort.

Navigation zur Zielfahne Platz für ein Bild Der nächste Zielpunkt ist mit einem Zielfähnchen auf der Karte markiert. Navigation mit Richtungspfeil Der Spieler wird mittels Richtungspfeil und Entfernungsangabe zum nächsten Zielpunkt geleitet. Die Karte wird ausgeblendet. Navigation mit Textanweisung Der Spieler wird mittels Textanweisung zum nächsten Zielpunkt geleitet. Themenaufgaben

Themenaufgaben sind ortsgebundene Aufgaben zum Kartenlesenlernen oder beliebigen anderen Themen.
